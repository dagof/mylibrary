A personal public library for learning purposes. It contains subjects, created by you, where they will store your preferred books. Selecting a subject, you will see the assigned books.

This is a functional example of a Website using these current and fantastic technologies:

Server Side

  • NodeJS: the motor
  • MongoDB: the memory, executing queries
  • ExpressJS: dispatching responses from client's requests, CORS
  • YUIDoc - JavaScript Documentation Tool: the name says it (planned)

Front-End

  • Angular: workflow, ui-router
  • Google Books API: JSON data
  • JQuery: DOM
  • Bootstrap: UI (don't pay attention to UX)
  • YUIDoc - JavaScript Documentation Tool: the name says it (planned)

It shows some how to's and other details about MongoDB and routing with AngularUI Router.

It's very simple. Look for books by taping the title and choose one from the list. This list come from Google's servers.

Click on any book to edit its details. This specific data is stored into our MongoDB database, not Google's. To edit your bookshelfs on Google Books, visit the Google Books API page. This info won't go to Google because I want explain you MongoDB boundaries and fonctionnalities (not all but enough to make functional this example).
